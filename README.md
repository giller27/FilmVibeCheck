# 🎬 FilmVibeCheck - Оновлена Структура

## 📋 Про проект

**FilmVibeCheck** - це веб-застосунок для вибору фільмів за вайбом. Структура проекту була **реорганізована** для кращої зручності редагування кожної сторінки окремо.

## 📁 Нова структура проекту

```
FilmVibeCheck/
├── index.html           # Головна сторінка з навігацією та модальним вікном
├── styles.css           # Глобальні стилі для всього сайту
├── script.js            # АРХІВ (більше не використовується)
│
├── pages/               # 📄 HTML файли для кожної сторінки
│   ├── feed.html        # Головна страница (Що подивитися сьогодні?)
│   ├── finder.html      # Вайб Пошук (Пошук за параметрами)
│   ├── vault.html       # Бібліотека (Вся база фільмів)
│   ├── compare.html     # Баттл фільмів (Порівняння двох фільмів)
│   ├── charts.html      # Топ-листи (Рейтинги от користувачів)
│   └── profile.html     # Мій профіль (Персональна статистика)
│
└── js/                  # 🔧 JavaScript модулі для логіки сторінок
    ├── shared.js       # Спільні функції, база даних, навігація, модальне вікно
    ├── feed.js         # Логіка сторінки "Головна"
    ├── finder.js       # Логіка сторінки "Вайб Пошук"
    ├── vault.js        # Логіка сторінки "Бібліотека"
    ├── compare.js      # Логіка сторінки "Баттл фільмів"
    ├── charts.js       # Логіка сторінки "Топ-листи"
    └── profile.js      # Логіка сторінки "Профіль"
```

## ✨ Переваги нової структури

✅ **Модульність** - Кожна сторінка в окремому файлі  
✅ **Легше редагування** - Не потрібно шукати код у величезному файлі  
✅ **Динамичне завантаження** - Сторінки завантажуються при натисненні кнопки  
✅ **Простіше додавання нових сторінок** - Просто створіть нові файли  
✅ **Кращий порядок коду** - Логіка розділена за функціяльністю

## 🎯 Як працює

1. **Завантаження** → `index.html` завантажує навігацію, модальне вікно та `js/shared.js`
2. **Клік по пункту меню** → Тригерується функція `navigateTo(pageId)`
3. **Динамичне завантаження** → Зі папки `pages/` завантажується HTML файл сторінки
4. **Завантаження логіки** → Зі папки `js/` завантажується JavaScript модуль сторінки
5. **Ініціалізація** → Виконується функція `init()` з JS модуля для налаштування сторінки

## 📚 Опис файлів

### `index.html`

- Головна структура сайту
- Навігаційна панель з посиланнями на сторінки
- Контейнер `#main-container` для динамичного завантаження контенту сторінок
- Модальне вікно для деталей фільмів (універсальне для всіх сторінок)
- Скрипт для ініціалізації при завантаженні сторінки

### `js/shared.js`

**Спільний код для всіх сторінок:**

- **База даних** - Масив `moviesDatabase` з усіма фільмами
- **Управління станом** - Об'єкт `state` для зберігання даних користувача
- **Навігація** - Функції `navigateTo()` та `loadPageContent()` для переміщення між сторінками
- **Модальне вікно** - Функції для відкриття/закриття модалі фільму
- **Rating система** - Функції для оцінювання фільмів
- **Карточки фільмів** - Функція `createMovieCard()` для генерування HTML карточок

### `pages/*.html`

**Кожен файл містить HTML розмітку однієї сторінки:**

- Заголовки розділу
- Контейнери для динамічного впровадження контенту
- Форми фільтрів та кнопки
- ID для елементів, на які повинен звертатися JavaScript

### `js/feed.js`, `js/finder.js`, і т.д.

**Кожний файл містить:**

- Функцію `init()` - виконується автоматично при завантаженні сторінки
- Функції для рендерування контенту
- Обробники подій (click, change, input, etc.)
- Логіка фільтрації та сортування

**Приклад структури JS модуля:**

```javascript
// ============================================
// PAGE NAME
// ============================================
function init() {
  // Код для ініціалізації сторінки
  // Встановлення слухачів подій
  // Рендерування контенту
}

function reinderContent() {
  // Функції для відображення даних
}

function handleEvent() {
  // Обробники подій
}
```

## 🚀 Як додати нову сторінку

1. **Створіть HTML файл** у папці `pages/` (наприклад, `pages/new-page.html`)

   ```html
   <div class="page-header">
     <h1>Назва Сторінки</h1>
   </div>
   <div class="page-content" id="content">
     <!-- Ваш контент тут -->
   </div>
   ```

2. **Створіть JavaScript файл** у папці `js/` (наприклад, `js/new-page.js`)

   ```javascript
   function init() {
     // Ініціалізація сторінки
     console.log("Сторінка завантажена!");
   }
   ```

3. **Додайте посилання у меню** в `index.html`
   ```html
   <li><a href="#" class="nav-link" data-page="new-page">Нова Сторінка</a></li>
   ```

Готово! Сторінка буде автоматично завантажуватися при натисненні на посилання.

## 💾 Дані користувача

Всі дані зберігаються в `localStorage`:

- Список переглянутих фільмів
- Список охочих подивитися
- Оцінки користувача для кожного фільму
- Налаштування фільтрів

## 🎨 CSS класи

**Глобальні класи в `styles.css`:**

- `.page` - Контейнер сторінки (не використовується, але зберігається для назад сумісності)
- `.movies-grid` - Grid для розташування карточок фільмів
- `.modal` - Модальне вікно
- `.btn-primary`, `.btn-secondary` - Кнопки
