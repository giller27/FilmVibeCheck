ІНСТРУКЦІЇ ДЛЯ РОЗРОБНИКА - FilmVibeCheck
==========================================

🎯 КОРОТКЕ ПОЯСНЕННЯ
---------------------
Структура проекту була ПЕРЕДІЛАНА на окремі файли для зручності.
Тепер кожна сторінка розташована у свому файлі:
- pages/feed.html, pages/finder.html, і т.д. - КОНТЕНТ сторінок
- js/feed.js, js/finder.js, і т.д. - ЛОГІКА сторінок
- js/shared.js - спільні для всіх функції (база даних, навігація)

⚡ ЩОБИ ЗМІНИТИ ВМІСТ СТОРІНКИ
------------------------------

1. РЕДАГУВАННЯ HTML (ПОКАЖЧИКИ):
   - Знайдіть файл в папці pages/ (наприклад, pages/feed.html)
   - Відредагуйте HTML розмітку
   - Питання: "Де я можу знайти кнопку 'Пошукати'?"
     Відповідь: У файлі pages/finder.html пошукайте <button ... id="applyFilters">

2. РЕДАГУВАННЯ ЛОГІКИ (ФУНКЦІОНАЛЬНІСТЬ):
   - Знайдіть файл в папці js/ (наприклад, js/feed.js)
   - Відредагуйте JavaScript функції
   - Питання: "Як змінити, скільки фільмів відображається на сторінці?"
     Відповідь: У файлі js/feed.js функції renderTrendingMovies() 
     змініть .slice(0, 6) на потрібну вам кількість

3. РЕДАГУВАННЯ СТИЛІВ:
   - Відредагуйте styles.css
   - Всі стилі глобальні для всього сайту
   - Питання: "Як змінити колір кнопок?"
     Відповідь: У styles.css пошукайте .btn-primary або .btn-secondary

🔄 ЯК СТОРІНКИ ЗАВАНТАЖУЮТЬСЯ
-----------------------------
1. Користувач клікає на "Вайб Пошук" в навігації
2. Функція navigateTo("finder") виконується (в shared.js)
3. Сторінка pages/finder.html завантажується динамічно
4. JavaScript файл js/finder.js також завантажується та виконується
5. Функція init() з finder.js інітіалізує сторінку (налаштовує кнопки, слухачів подій, тощо)

➕ ЯК ДОДАТИ НОВУ СТОРІНКУ
---------------------------
1. Створіть файл pages/my-page.html
2. Напишіть HTML розмітку для сторінки
3. Створіть файл js/my-page.js
4. Напишіть функцію init() для логіки
5. Іде додайте посилання в index.html:
   <li><a href="#" class="nav-link" data-page="my-page">Моя Сторінка</a></li>

Готово! Сторінка буде автоматично завантажуватися при клікові.

🐛 ВІДЛАДКА
-----------
- Відкрийте DevTools (F12) та перевірте консоль для помилок
- Перевірте Network tab щоб переконатися, що файли завантажуються
- Переконайтесь, що функція init() визначена в js файлі

💾 ДАНІ КОРИСТУВАЧА
-------------------
Всі дані зберігаються в localStorage браузера:
- Переглянуті фільми
- Список для перегляду
- Оцінки користувача

🎨 КЛАСИ CSS, ЯКІ ЧАСТО ВИКОРИСТОВУЮТЬСЯ
------------------------------------------
- .movie-card          - Карточка фільму
- .modal               - Модальне вікно
- .btn-primary         - Первинна кнопка
- .btn-secondary       - Вторинна кнопка
- .movies-grid         - Grid контейнер для карточок
- .nav-link            - Посилання у навігації

📞 ЧАСТОВІ ВОПРОСИ
------------------
В: Чому я повинен більше редагувати в index.html?
О: Всі сторінки завантажуються динамічно. Основної контент в pages/ папці.

В: Як отримати доступ до moviesDatabase?
О: Він визначений в js/shared.js і доступний всім іншим скриптам.

В: Де зберігаються дані користувача?
О: В localStorage можна знайти об'єкт 'vibecheckState'.

В: Як я можу тестувати нову сторінку?
О: Просто відкрийте index.html в браузері і натисніть на посилання.
